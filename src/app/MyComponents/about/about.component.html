My Todo List App

A Todo App for Storing Daily Task.

In this app angluar is used for making one page application and main feature is that the page does not get loaded here only the content changes in this app.
Features of App

  • You Can Add a Task
  • Your Can Delete a Task
  • Mark as Task Completed
  • Your Task are Stroed in Browser localstorage

Features of Angluar

    It uses the Angular to store the todo app
    It make use of the 
    Routing Module
    ng-for
    *ngIf=expression
    EventEmitter
    and Bootstrap Jquery
